# Architecture Decision Record (ADR) — Проєкт Vox

## 1. Контекст
Мета проєкту — створення hands-free інструменту для розпізнавання мови та автоматичного введення тексту (інжекції) на Linux (X11) та Windows.

## 2. Прийняті рішення
- **STT Engine:** Використовується **faster-whisper** (серверна частина) для максимальної точності розпізнавання української мови.
- **WebSocket Streaming:** Реалізовано потокову передачу аудіо через WebSockets для мінімізації затримки (latency).
- **Deployment:** Сервер винесено на окремий малопотужний пристрій (Mini-PC Intel N150) для уникнення лагів на робочій машині.
- **Інжекція тексту:** Використовується гібридний метод (X11: швидка вставка через `xdotool` + буфер обміну; Windows: API інжекція).
- **Vosk:** Усі спроби реалізації локального Vosk були визнані невдалими через недостатню точність маленьких моделей та технічні обмеження реалізації. Рішення: **Vosk повністю видалено з проєкту**.

## 3. Статус
Активна розробка та підтримка Whisper-based архітектури.
