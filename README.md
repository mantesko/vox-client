# Vox — Hands-Free Voice-to-Text Utility 🎙️⌨️

**Vox** — це глобальна фонова hands-free утиліта для введення тексту та команд голосом у будь-яке активне вікно системи (термінал, редактор коду, месенджер тощо). 
Вона оптимізована для розробників, які хочуть працювати без рук, та повністю підтримує українську мову.

Проєкт використовує клієнт-серверну архітектуру:
*   **Сервер (Backend):** Окремий репозиторій [vox-server](https://github.com/mantesko/vox-server). Працює в Docker, використовує `faster-whisper`.
*   **Клієнт (Desktop Client):** Цей репозиторій. Фоновий демон на Python з системним треєм, красивим напівпрозорим оверлеєм та підтримкою симуляції введення (X11/Wayland/Windows).

---

## 🏛️ Архітектура (DDD)

Проєкт побудований за принципами **Domain-Driven Design**, що забезпечує чіткий поділ відповідальності:

*   **`domain/`**: Ядро системи. Містить сутності (Entities), інтерфейси репозиторіїв та доменну логіку, що не залежить від зовнішніх бібліотек.
*   **`application/`**: Шар оркестрації. Містить сервіси додатку, які координують роботу доменних сутностей та DTO для передачі даних.
*   **`infrastructure/`**: Реалізація інтерфейсів. Тут знаходиться код для роботи з мікрофоном (`SoundDeviceMicrophone.py`) та інжекції тексту в систему (`X11TextInjector.py`).
*   **`presentation/`**: Графічний інтерфейс. Містить оверлей (`TranscriptionOverlay.py`) та іконку в треї (`VoxTrayIcon.py`).
*   **`vox_client.py`**: Точка входу в додаток.

## 💻 Налаштування та запуск Клієнта

### 1. Встановлення залежностей
Перейдіть у папку проекту та створіть віртуальне середовище Python:
```bash
# Створюємо віртуальне оточення
python3 -m venv venv

# Активуйте його
source venv/bin/activate

# Встановлюємо залежності
pip install -r requirements.txt
```

### 2. Конфігурація
Відкрийте файл `config.py` або налаштуйте змінні оточення у файлі `.env`:
```bash
VOX_API_KEY=твій_секретний_ключ
VOX_SERVER_PROFILE=remote  # або local
```

### 3. Запуск
```bash
source venv/bin/activate
python3 vox_client.py
```

## 🕹️ Як користуватися Vox

1. **Запуск у фоні:** У треї з'явиться іконка:
   * **Сірий:** Очікування.
   * **Зелений/Синій:** Слухає фразу "привіт вокс".
   * **Червоний:** Запис та транскрибація.
2. **Активація:** Голосом (**«привіт вокс»**) або гарячою клавішею (**`Ctrl + Space`**).
3. **Диктовка:** Диктуйте текст, оверлей покаже результат. Vox автоматично вставить текст та натисне `Enter` після завершення.
